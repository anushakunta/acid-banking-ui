# •	FinTrust Bank Frontend — React + Redux + MUI + Auth + Charts

## Run
```bash
npm install
npm run dev
```
Backend expected at `http://localhost:3000/graphql`.

## Pages
- Login / Signup
- Dashboard (Chart.js line chart of balances)
- Accounts (list + create transactions)

## State
- Classic Redux (no Toolkit), Thunk for async
- Auth token persisted in localStorage
